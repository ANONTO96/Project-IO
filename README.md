# Project IO

Project IO is a web application designed to showcase all of our web development projects in one place.  
It works as a portfolio-style platform where different types of web apps we have built are displayed along with the technologies used, project descriptions, and other relevant details.

The goal of Project IO is to organize and present our development work clearly so that visitors can easily explore our projects, the stacks we used, and the ideas behind each application.

## Features

- Display a collection of all developed web applications
- Detailed view of each project
- Technology stack used for every project
- Clean and responsive UI
- Alert notifications for user actions
- Easy navigation between project pages

## Technologies Used

This project was built using the following tools and libraries:

- **React** – Frontend library for building UI
- **React Router DOM** – Client-side routing
- **Tailwind CSS** – Utility-first CSS framework
- **DaisyUI** – Tailwind CSS component library
- **React Icons** – Icon library for UI elements
- **React Toastify** – Toast notifications and alerts
- **Vite** – Fast frontend build tool and development server

## Purpose

Project IO acts as a centralized hub for showcasing our web development work.  
It helps organize multiple projects in one interface and demonstrates the technologies and approaches used in each application.
